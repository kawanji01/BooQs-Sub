# BooQs-Sub
English | [日本語](./README-ja.md)
 
BooQs-Sub is a tool to download captions from YouTube.

With BooQs-Sub, You can download captions as SRT, CSV, and TXT.

Check it out => https://www.booqs-sub.com/en
